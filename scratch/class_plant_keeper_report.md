## 1. 연구의 필요성 및 장점

### 가. 개발의 필요성
1. **교실 내 식물 관찰 교육의 시공간적 한계 극복**: 초등학교 과학 교육과정의 '식물의 한살이' 단원은 식물의 일생을 관찰하며 생명 존중 및 과학적 탐구 역량을 기르는 핵심 단원입니다. 그러나 교실 창가의 부족한 조도, 주말 및 방학 중 물 주기 단절, 예상치 못한 병해로 인한 식물 고사 등으로 인해 학생들이 온전한 관찰 사이클을 완성하지 못하는 경우가 빈번합니다.
2. **관찰 일지 기록의 양방향 디지털화(DX)**: 종이에 작성하는 관찰 일지는 관찰한 사진을 현상하여 붙이기 번거롭고, 반려식물의 성장 수치(키, 잎의 개수) 변화를 시각화하여 파악하기에 한계가 있습니다. 음성 인식(STT)과 클라우드 기반 성장 선형 차트를 도입하여 글쓰기 장벽을 낮추고 성장을 한눈에 추적하는 디지털 환경이 필요합니다.
3. **독립적이고 안전한 학교용 클라우드 아키텍처 요구**: 기존 상용 플랫폼은 학교 예산 문제나 개발사 사정에 따라 서버 서비스가 중단되면 축적된 아동들의 관찰 일지가 유실되는 심각한 문제가 있었습니다. 교사 본인의 Supabase 무료 서버 계정을 연동하는 **BYOD(Bring Your Own Database)** 구조를 채택함으로써, 외부 상용 솔루션의 예산 장벽을 허물고 학급 전용 데이터를 영구적으로 안전하게 자율 통제할 수 있는 시스템이 필요합니다.

### 나. 「우리 반 식집사」 웹 앱의 주요 장점
1. **다교과 융합(Cross-curricular) 교육 모듈 탑재**: 반려식물 재배(과학·실과)와 이미지 알고리즘 블록 배치(정보), 음성 인식과 질문 가이드를 결합한 생생한 글쓰기(국어)가 하나의 웹 앱 안에서 유기적으로 연동되어 종합적 융합 학습을 가능케 합니다.
2. **뛰어난 모바일 반응형 이중 스케일링 레이아웃**: 스마트폰의 카메라를 들고 식물 앞에서 즉시 관찰 일지를 쓸 수 있도록 최적화되었습니다. 좁은 해상도에서도 도구 변별 카드가 겹치지 않는 3열 그리드 자동 전환과 이미지 코딩 원형 링의 컴포넌트 반경 축소(95px → 65px), 그리고 세로 짤림을 막기 위한 최적의 스케일 가공(`max-h-[92vh]`)으로 스크롤 및 잘림 문제를 완벽히 해결했습니다.
3. **서버 운영 비용 제로화 및 1초 인프라 활성화**: 교사용 대시보드에 내장된 '1초 테이블 마법 셋업'을 통해, 교사가 데이터베이스 테이블 설계 지식 없이 제공된 SQL을 1클릭 복사 및 실행하는 것만으로 학생들이 바로 접근 가능한 전용 클라우드 데이터 서버를 즉시 확보할 수 있습니다.

## 3. 소프트웨어의 실제 구현 기능 (유목화 분석)

### 가. 교사 대시보드 (Teacher Dashboard) 실제 구현 기능

| 기능 분류 | 상세 구현 기능 | 상세 설명 | 사용 기술 및 연동 데이터 |

### 나. 학생 대시보드 (Student Dashboard) 실제 구현 기능

| 기능 분류 | 상세 구현 기능 | 상세 설명 | 사용 기술 및 연동 데이터 |
| :---: | :--- | :--- | :--- |
| **1** | **식집사 개인화 헤더** | 로그인한 학생의 성별 정보(`gender` 컬럼)에 맞춰 여학생/남학생 캐릭터 일러스트가 아기자기한 플로팅 애니메이션과 함께 노출. D-Day 및 학급 정보 연동. | `students` 테이블 내 아동 성별 대조 및 CSS Float 모션 |

| **8** | **AI 식물 이상 진단** | 식물이 아플 때 잎의 변색, 시듦, 벌레 발생 등 초등 다빈도 10가지 증상 중 하나를 선택하면 해당 병해의 원인과 AI 처방전(수분, 습도, 광량 대처법)을 즉시 반환. | 클라이언트 내장 진단 마크업 및 React 애니메이션 노출 |
| **9** | **식물 가꾸기 마스터<br>(Step 1 ~ Step 4)** | 1단계: 필수 도구 찾기 (3열 격자)<br>2단계: 식물 심기 알고리즘 코딩 링 (반경 65px 배치 및 순서 체크)<br>3단계: 환경 학습 미션<br>4단계: 폭죽 애니메이션을 포함한 성공 축하 피드백. | canvas-confetti 라이브러리 연동 및 6단계 순서 검증 알고리즘 |

## 5. 결론 및 제언

### 가. 결론
1. **아동 친화적 탐구 인터페이스 실효성 입증**: 「우리 반 식집사」 플랫폼은 모바일 기기 및 스마트폰의 가로 해상도에 구애받지 않고, Step 1.5 가로 배치 전환 및 Step 2 코딩 반경의 모바일 다운 스케일링을 통해 상하단이 잘리는 현상을 성공적으로 극복하여 현장 중심형 에듀테크로서의 완벽한 이중 적응형 인터페이스를 수립했습니다.
2. **독립적이고 투명한 데이터 생태계 구현**: 상용 서버 비용 부담이나 아동의 민감정보 유출 걱정 없이, 교사가 Supabase 클라우드 인프라를 활용하여 학급 전용 서버(BYOD)를 무료로 무한 운영하는 인프라 안전망을 성공적으로 연계했습니다.

### 나. 제언
1. **데이터 업로드 효율성 제고**: 학교 현장의 네트워크 트래픽 과부하를 고려하여, 사진 파일 업로드 시 클라이언트 사이드 단에서 base64 압축 리사이징 처리를 하고 있으나, 저사양 태블릿에서의 성능 향상을 위한 데이터 로딩 지연 현상의 정기적 모니터링을 제언합니다.
2. **식물 도감 연계 확장**: 초등학생의 식물 탐구욕을 더욱 자극할 수 있도록 현재 탑재된 10가지 식물 이상 증상 외에, 학급에서 많이 기르는 대표 작물들의 특징을 담은 미니 데이터 사전 등을 추가하는 확장 연구를 제언합니다.
